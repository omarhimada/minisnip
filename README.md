# minisnip
- Copy to clipboard without requiring `Windows Clipboard Service` (`services.msc`).
- Includes `auto_minisnip.exe` bound to `Ctrl` + `Alt `+ `N` - compiled with AHK.
- The executable itself `minisnip.exe` takes one screenshot then immediately exits.
- That executable is 150 MB because it isn't .NET Framework dependent, all requirements are included in the executable.
